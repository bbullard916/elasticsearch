# Cases Folder

Store per-case working files here.

Suggested structure per case:
- `cases/<CASE_ID>/intake.md` (copied from `templates/case_intake.md`)
- `cases/<CASE_ID>/evidence/` (logs, API outputs, screenshots)
- `cases/<CASE_ID>/triage_output.md` (copied from `templates/triage_output.md`)

Keep sensitive customer data redacted where required.
